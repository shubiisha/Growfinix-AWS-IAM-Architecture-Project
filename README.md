# AWS IAM Architecture Project

## Overview

Designed a secure IAM architecture for a mock company using AWS IAM.

## Technologies Used

- AWS IAM
- AWS Billing
- Amazon EC2
- IAM Policies

## Roles Implemented

|Role|Access|
|-|-|
|Developer|EC2 management|
|Billing|Billing read-only|
|Auditor|Read-only auditing|
|Administrator|Full access|

## Security Features

- Least privilege access
- IAM groups
- MFA authentication
- Password policy
- Permission review

## Project Outcome

Implemented secure role-based access control following AWS IAM best practices.
